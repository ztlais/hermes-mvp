# Méthodologie : Stages de développement projet solaire

## Contexte

Cette taxonomie définit les **8 stages** par lesquels passe un projet solaire / EnR, de l'origination jusqu'à la construction. Chaque stage décrit l'avancement sur 7 dimensions clés :

| Dimension | Abréviation |
|---|---|
| Real Estate (Foncier) | RE |
| Environmental (Environnement) | Env |
| Authorities (Autorités) | Auth |
| Easements (Servitudes) | Easements |
| Permitting (Permis) | Permitting |
| Grid Connection (Raccordement) | Grid |
| Tariff (Tarif) | Tariff |

---

## 1. Origination
**Rien de sécurisé.**
| Dimension | Statut |
|---|---|
| Real Estate | Possible |
| Authorities | Consulted |
| Grid Connection | Consulted |

> Le projet vient d'être identifié. Aucun terrain signé, aucune démarche engagée. Simple prospection.

---

## 2. Early
| Dimension | Statut |
|---|---|
| Real Estate | **Secured** |
| Environmental | Ongoing |
| Authorities | Consulted |
| Easements | Consulted |
| Grid Connection | Available |

> Le foncier est sécurisé (promesse de bail ou option). Les études environnementales commencent. Permis pas encore déposé.

---

## 3. Submit
| Dimension | Statut |
|---|---|
| Real Estate | Secured |
| Environmental | **Done** |
| Authorities | **Secured** |
| Easements | Possible |
| Permitting | **Near apply** |
| Grid Connection | Available |

> Les études sont terminées, les autorisations obtenues. Le dépôt du permis est imminent.

---

## 4. Mid
| Dimension | Statut |
|---|---|
| Real Estate | Secured |
| Environmental | Done |
| Authorities | Secured |
| Easements | **Secured** |
| Permitting | **Submitted** |
| Grid Connection | Available |

> Le permis de construire a été déposé. On attend l'instruction.

---

## 5. Advanced
| Dimension | Statut |
|---|---|
| Real Estate | Secured |
| Environmental | Done |
| Authorities | Secured |
| Easements | Secured |
| Permitting | **Deemed completed** |
| Grid Connection | Available |

> Le permis est considéré comme complet (instruction terminée). En attente de la décision finale ou du jugement.

---

## 6. Nearly secured
| Dimension | Statut |
|---|---|
| Real Estate | Secured |
| Environmental | Done |
| Authorities | Secured |
| Easements | Secured |
| Permitting | **Obtained** |
| Grid Connection | **Nearly secured** |
| Tariff | **Nearly secured** |

> Le permis est obtenu. Le raccordement et le tarif sont presque finalisés. Dernière ligne droite avant le "prêt à construire".

---

## 7. Secured & clean of appeal
| Dimension | Statut |
|---|---|
| Real Estate | Secured |
| Environmental | Done |
| Authorities | Secured |
| Easements | Secured |
| Permitting | **Obtained** |
| Grid Connection | **Secured** |
| Tariff | **Secured** |

> **RTB (Ready To Build).** Tout est sécurisé, permis purgé (délai de recours passé), raccordement et tarif signés. Le projet est prêt à être construit.

---

## 8. Refused
| Dimension | Statut |
|---|---|
| Permitting | **❌ Refused** |

> Le projet est arrêté — refus de permis définitif.

---

## Résumé visuel

| Stage | RE | Env | Auth | Easements | Permitting | Grid | Tariff |
|---|---|---|---|---|---|---|---|
| Origination | Possible | — | Consulted | — | — | Consulted | — |
| Early | **Secured** | Ongoing | Consulted | Consulted | — | Available | — |
| Submit | Secured | **Done** | **Secured** | Possible | **Near apply** | Available | — |
| Mid | Secured | Done | Secured | **Secured** | **Submitted** | Available | — |
| Advanced | Secured | Done | Secured | Secured | **Deemed completed** | Available | — |
| Nearly secured | Secured | Done | Secured | Secured | **Obtained** | **Nearly secured** | **Nearly secured** |
| Secured & clean | Secured | Done | Secured | Secured | Obtained | **Secured** | **Secured** |
| Refused | — | — | — | — | **❌ Refused** | — | — |

---

## Notes
- **Real Estate (RE)** est toujours le premier à être sécurisé (dès Early).
- **Environmental (Env)** est toujours "Done" à partir de Submit.
- **Permitting** est le grand différenciateur : Near apply → Submitted → Deemed completed → Obtained.
- **Grid & Tariff** ne se sécurisent qu'à la fin (Nearly secured puis Secured).
- Un projet peut cumuler plusieurs dimensions à des stades différents (ex: RE Secured + Env encore Ongoing = Early).
